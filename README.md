# Парсер политических партий

Программа для парсинга списка политических партий с сайта Министерства юстиции РФ.

## Функциональность

- Парсит HTML-страницу со списком политических партий
- Извлекает названия партий и ссылки на документы (устав, регистрационные сведения)
- Сохраняет результат в JSON формате
- Обрабатывает относительные ссылки, делает их абсолютными
- Корректно отмечает отсутствие документов

## Установка

```bash
pip install -r requirements.txt
```
ИСПОЛЬЗОВАНИЕ

1. Скачайте страницу вручную: https://minjust.gov.ru/ru/pages/politicheskie-partii/

2. Сохраните как 'Веб-страница, полностью' в формате HTML

3. Запустите парсер:

python parse_parties.py

4. Программа найдет HTML файл и создаст файл с результатами

РЕЗУЛЬТАТ

Программа создает файл parties.json со структурой:

[
  {
    "name": "Название политической партии",
    "doc_url": "https://minjust.gov.ru/.../document.pdf"
  }
]

Если документ не найден, поле doc_url будет содержать null.
